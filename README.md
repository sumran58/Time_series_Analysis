## Folder Description

### practice/

This folder contains all practical implementations and notebooks related to time series analysis.

### decomposition/

Contains notebooks demonstrating **time series decomposition**, including:

* Trend component
* Seasonal component
* Residual component
* Additive decomposition

### stationarity/

Contains notebooks related to **stationarity testing**, including:

* Rolling statistics
* Augmented Dickey-Fuller (ADF) Test
* KPSS Test

### autocorrelation/

Contains notebooks explaining:

* Autocorrelation
* Partial Autocorrelation
* ACF and PACF plots

### forecasting/

Contains notebooks implementing forecasting techniques such as:

* AR models
* MA models
* ARIMA models
